# ReactionTimeTest

Небольшой тест на скорость реакции на Python и Pygame. Нажимайте мышью только после того, как экран станет зеленым, и программа покажет время реакции в миллисекундах.

## Возможности

- окно с инструкциями и коротким циклом теста
- случайная задержка перед сменой цвета
- измерение времени реакции и отображение результата
- отдельный экран при слишком раннем клике

## Требования

- Python 3.13+
- Pygame 2.6+

## Установка

Создайте виртуальное окружение и установите зависимости:

```bash
uv sync
uv run run.py
```

## Запуск

```bash
uv run run.py
```

## Управление

- клик мышью — начать тест
- после смены экрана на зеленый — клик для фиксации реакции
- закрытие окна — выход

## Структура проекта

- `run.py` — основной файл запуска и игровой цикл
- `screens.py` — отрисовка экранов и текстов
- `config.py` — цвета и размеры окна

## Как это работает

После старта показывается синий экран, клик запускает тест. Появляется красный экран с ожиданием, затем через случайную задержку — зеленый. Время реакции измеряется от смены на зеленый до клика мышью.
